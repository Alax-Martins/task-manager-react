# 🚀 Task Manager React - Portfolio Dev Junior

Um **Task Manager moderno** construído com **React 18 + Vite + Tailwind CSS** e **localStorage** para persistência de dados.

## ✨ Funcionalidades
- ✅ Adicionar novas tasks (Enter ou botão)
- ✅ Marcar tasks como concluídas (checkbox ou clique no texto)
- ✅ Deletar tasks individuais
- 💾 Persistência automática com localStorage
- 📱 Design responsivo e moderno com gradientes
- 🎨 UI/UX polida com animações suaves
- 📊 Contador de tasks concluídas

## 🛠️ Tech Stack
- **React 18** (Hooks: useState, useEffect)
- **Vite** (build ultra-rápido)
- **Tailwind CSS** (estilização utility-first)
- **localStorage API** (persistência no browser)

## 🚀 Como Executar

```bash
npm install
npm run dev
```

Abra http://localhost:5173
